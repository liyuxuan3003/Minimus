# Minimus

Minimus是个人在长期LaTeX写作中整理出的一套宏包，包含许多自定义命令和环境，并包含了众多常用的宏包。

Minimus的名称来源于坎巴拉太空计划（Kerbal Space Program）中对应现实中地球的Kerbin的第二卫星Minmus的错拼。

Minimus项目目前定义的宏包列表如下，使用时请遵照该顺序，调换顺序可能导致错误
- `minimus-section`
- `minimus-text`
- `minimus-math`
- `minimus-float`
- `minimus-code`
- `minimus-colorbox`
- `minimus-reference`
- `minimus-bibtex`
